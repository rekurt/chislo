Why chislo

Russian numerals are deceptively hard. chislo handles the grammar so you don't have to.

⚖️

Grammatically correct

Three genders (один / одна / одно), full noun declension, and proper decimal agreement (одна целая, not один целых).

🧾

Built for documents

Money, percentages, durations, dates & times, and fractions — everything receipts, invoices and payment orders need.

💱

11 currencies

RUB, USD, EUR, GBP, CNY, JPY, KZT, BYN, UAH, CHF, AED out of the box, plus ISO 4217 lookup and custom currencies.

⚡

Zero-alloc dictionary

All data lives in const; fluent Display wrappers format straight into write! with no extra String.

🪶

Tiny & portable

no_std with alloc, optional rust_decimal, and WASM bindings via wasm-bindgen.

🦀

i64 range

Integers up to quintillions; the scale dictionary reaches 10³⁹ (duodecillions).

One look, and you get it

Add the crate and start converting.

use chislo::{int_to_words, decline, ordinal, money, date_to_words, Gender, RUB};

int_to_words(42);                          // "сорок два"
decline(5, "рубль", "рубля", "рублей");   // "рублей"
ordinal(2026, Gender::Masculine);        // "две тысячи двадцать шестой"
money(1234, 56, &RUB);
// "одна тысяча двести тридцать четыре рубля пятьдесят шесть копеек"
date_to_words(2026, 4, 10);                // "десятое апреля две тысячи двадцать шестого года"
