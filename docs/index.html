chislo · числа прописью

Превращает любое число в правильные русские слова — род, склонение существительных, валюты, проценты, дроби, длительность, даты и время. Быстрый, нетребовательный к зависимостям крейт на Rust.

▶ Попробовать вживую 📖 Документация API 🦀 rustdoc ★ GitHub
cargo add chislo no_std · WASM · словарь без аллокаций

Почему chislo

Русские числительные обманчиво сложны. chislo берёт грамматику на себя.

⚖️

Грамматически верно

Три рода (один / одна / одно), полное склонение существительных и правильное согласование десятичных (одна целая, а не один целых).

🧾

Создано для документов

Деньги, проценты, длительность, даты и время, дроби — всё, что нужно чекам, счетам и платёжным поручениям.

💱

11 валют

RUB, USD, EUR, GBP, CNY, JPY, KZT, BYN, UAH, CHF, AED из коробки, плюс поиск по ISO 4217 и свои валюты.

⚡

Словарь без аллокаций

Все данные живут в const; плавные обёртки Display пишут прямо в write! без лишней String.

🪶

Компактна и портативна

no_std с alloc, опциональный rust_decimal и WASM-биндинги через wasm-bindgen.

🦀

Диапазон i64

Целые до квинтиллионов; словарь разрядов доходит до 10³⁹ (дуодециллионы).

Один взгляд — и всё понятно

Добавьте крейт и начните преобразовывать числа.

use chislo::{int_to_words, decline, ordinal, money, percent, fraction, duration_hms, date_to_words, Gender, RUB};

int_to_words(42);                          // "сорок два"
decline(5, "рубль", "рубля", "рублей");   // "рублей"
ordinal(2026, Gender::Masculine);        // "две тысячи двадцать шестой"
money(1234, 56, &RUB);                  // "…тридцать четыре рубля пятьдесят шесть копеек"
percent(42);                          // "сорок два процента"
fraction(1, 2);                        // "одна вторая"
duration_hms(1, 30, 0);                // "один час тридцать минут"
date_to_words(2026, 4, 10);            // "десятое апреля две тысячи двадцать шестого года"
